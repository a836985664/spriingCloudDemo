# 微信扫码登录功能测试说明

## 📋 功能概述

实现了微信扫码登录功能，支持：
- **PC端**：扫码登录
- **手机端**：点击登录

## 🔧 接口说明

### 1. 获取二维码（PC端）
```
GET /api/auth/wechat/qrcode
```

**响应：**
```json
{
  "qrcodeUrl": "https://mock-wechat.com/qrcode?state=xxx",
  "qrcodeImage": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...",
  "state": "unique_state_123"
}
```

**说明：**
- `qrcodeUrl`: 二维码内容URL（前端可用此URL生成二维码）
- `qrcodeImage`: 二维码图片（base64格式，前端可直接显示）
- `state`: 状态标识（用于轮询）

---

### 2. 查询登录状态（PC端轮询）
```
GET /api/auth/wechat/status?state=xxx
```

**响应：**
```json
{
  "status": "pending",  // pending-待扫码，success-成功，expired-过期
  "token": "jwt_token_..."  // 登录成功时返回
}
```

---

### 3. 获取微信授权URL（手机端）
```
GET /api/auth/wechat/auth-url
```

**响应：**
```json
{
  "authUrl": "https://mock-wechat.com/connect/oauth2/authorize?..."
}
```

---

### 4. 微信回调（两种登录方式都用）
```
GET /api/auth/wechat/callback?code=xxx&state=xxx
```

**响应：**
```json
{
  "token": "jwt_token_...",
  "userInfo": {
    "id": 1,
    "username": "微信用户_mock_ope",
    "nickname": "微信用户",
    "avatar": "https://mock-avatar.com/default.png"
  }
}
```

---

### 5. 模拟扫码（测试用）
```
POST /api/auth/wechat/mock-scan
Content-Type: application/json

{
  "state": "unique_state_123"
}
```

**响应：**
```json
"jwt_token_..."
```

## 🎯 前端集成示例

### PC端前端代码

```javascript
// 1. 获取二维码
const { qrcodeUrl, qrcodeImage, state } = await fetch('/api/auth/wechat/qrcode').then(r => r.json());

// 2. 显示二维码（两种方式）

// 方式1：直接显示base64图片（推荐）
document.getElementById('qrcode').src = qrcodeImage;

// 方式2：使用QRCode库生成二维码
// 需要先引入 qrcode.js 库
// new QRCode(document.getElementById('qrcode'), qrcodeUrl);

// 3. 轮询登录状态
const pollInterval = setInterval(async () => {
    const response = await fetch(`/api/auth/wechat/status?state=${state}`).then(r => r.json());

    if (response.status === 'success') {
        clearInterval(pollInterval);
        const token = response.token;
        // 保存token，跳转到首页
        localStorage.setItem('token', token);
        window.location.href = '/home';
    } else if (response.status === 'expired') {
        clearInterval(pollInterval);
        alert('二维码已过期，请刷新页面重新获取');
    }
}, 2000); // 每2秒轮询一次
```

### 手机端前端代码

```javascript
// 1. 获取授权URL
const { authUrl } = await fetch('/api/auth/wechat/auth-url').then(r => r.json());

// 2. 跳转到微信授权页面
window.location.href = authUrl;

// 3. 微信回调后，从URL参数获取token
const urlParams = new URLSearchParams(window.location.search);
const token = urlParams.get('token');

if (token) {
    // 保存token，跳转到首页
    localStorage.setItem('token', token);
    window.location.href = '/home';
}
```

## 📱 设备判断示例

```javascript
// 判断是否为移动设备
function isMobile() {
    return /Android|webOS|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(navigator.userAgent) ||
           window.innerWidth < 768;
}

// 根据设备类型显示不同UI
if (isMobile()) {
    // 显示"微信登录"按钮
    document.getElementById('mobile-login').style.display = 'block';
} else {
    // 显示二维码
    document.getElementById('pc-login').style.display = 'block';
}
```

## ⚠️ 注意事项

1. **二维码有效期**：2分钟
2. **轮询频率**：建议每2秒轮询一次
3. **过期处理**：二维码过期后需要重新获取
4. **模拟数据**：当前使用模拟数据，后续需要替换为真实微信API
