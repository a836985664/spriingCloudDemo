# 微信扫码登录功能更新说明

## 📝 更新内容

### 1. 添加二维码图片生成功能

**问题：**
- 之前只返回二维码URL，前端需要自己生成二维码图片

**解决方案：**
- 添加 ZXing 二维码生成库
- 后端直接生成二维码图片（base64格式）
- 前端可以直接显示图片，无需额外处理

### 3. 修改 Service

**WechatAuthService.java**

添加 `generateQrcodeImage()` 方法：
```java
/**
 * 生成二维码图片（base64格式）
 */
private String generateQrcodeImage(String content) {
    try {
        // 创建二维码生成器
        QRCodeWriter qrCodeWriter = new QRCodeWriter();
        BitMatrix bitMatrix = qrCodeWriter.encode(
                content,
                BarcodeFormat.QR_CODE,
                300, // 宽度
                300, // 高度
                Map.of(
                        EncodeHintType.CHARACTER_SET, "UTF-8",
                        EncodeHintType.MARGIN, 1
                )
        );

        // 转换为BufferedImage
        BufferedImage image = new BufferedImage(
                bitMatrix.getWidth(),
                bitMatrix.getHeight(),
                BufferedImage.TYPE_INT_RGB
        );

        for (int x = 0; x < bitMatrix.getWidth(); x++) {
            for (int y = 0; y < bitMatrix.getHeight(); y++) {
                image.setRGB(x, y, bitMatrix.get(x, y) ? 0xFF000000 : 0xFFFFFFFF);
            }
        }

        // 转换为base64
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        ImageIO.write(image, "PNG", baos);
        byte[] imageBytes = baos.toByteArray();
        String base64Image = Base64.getEncoder().encodeToString(imageBytes);

        return "data:image/png;base64," + base64Image;

    } catch (Exception e) {
        log.error("生成二维码图片失败", e);
        return null;
    }
}
```

修改 `generateQrcode()` 方法：
```java
public QrcodeResponse generateQrcode() {
    String state = UUID.randomUUID().toString();
    long createTime = System.currentTimeMillis();

    // 模拟微信二维码URL
    String qrcodeUrl = "https://mock-wechat.com/qrcode?state=" + state;

    // 生成二维码图片（base64）
    String qrcodeImage = generateQrcodeImage(qrcodeUrl);

    // 存储登录状态
    LoginState loginState = new LoginState(state, "pending", null, null, createTime);
    stateMap.put(state, loginState);

    log.info("生成二维码，state: {}", state);

    return new QrcodeResponse(qrcodeUrl, qrcodeImage, state);
}
```

## 🎯 前端使用方式

### 方式1：直接显示base64图片（推荐）
```javascript
const { qrcodeImage, state } = await fetch('/api/auth/wechat/qrcode').then(r => r.json());
document.getElementById('qrcode').src = qrcodeImage;
```

### 方式2：使用QRCode库生成二维码
```javascript
const { qrcodeUrl, state } = await fetch('/api/auth/wechat/qrcode').then(r => r.json());
new QRCode(document.getElementById('qrcode'), qrcodeUrl);
```
